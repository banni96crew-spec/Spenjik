PRICE CONTRACT:

Используй M2 card catalogs/definitions. Не дублируй изменяемую price table и не парси display text.

Scaling:

- Informant: 5 / 6 / 7 при owned 0 / 1 / 2+.
- Laundry: 8 / 10 / 12 при owned 0 / 1 / 2+.

Protected Nal:

- 0 Accountants → 0;
- 1 → 4;
- 2 → 6;
- N > 2 → `6 + (N - 2)`.

Modifier order:

1. Base/scaled price.
2. Rebuild override.
3. Role hook.
4. Turf hook.
5. Temporary/contact modifiers.
6. Clamp к minimum 1.

Price preview не мутирует state, random, flags, modifiers или Resources.

Role/Turf owner modules ещё не реализованы. Создай clean integration contract, но не дублируй их полную логику внутри M6.

MARKET CONTRACT:

Always available:

- informant, stash, thug, cops.

Rotating pool:

- laundry, accountant, brothel, workshop;
- district_control, cartel, judge;
- bruiser, cleaner, insider, saboteur, federal_raid.

Rules:

- pools не пересекаются;
- IDs valid и unique;
- 4 rotating slots ниже Turf Level 4, иначе 3;
- selection через `SeededPicker.pick_unique`;
- shared random state обновляется exact selected count steps;
- same initial state → same market;
- market соответствует M4 schema.

PURCHASE CONTRACT:

Validation проверяет:

- Market phase;
- valid state/player/card;
- availability;
- enough Nal по final price;
- one copy одного card ID на player за round;
- card-specific requirements и limits.

Используй documented errors:

- `INVALID_PHASE`, `INVALID_PLAYER_ID`, `INVALID_CARD_ID`;
- `CARD_NOT_AVAILABLE_IN_MARKET`, `NOT_ENOUGH_NAL`;
- `CARD_ALREADY_PURCHASED_THIS_ROUND`;
- `REQUIREMENT_NOT_MET`, `CARD_LIMIT_REACHED`, `INVALID_STATE`.

Missing required error — PRD/OQ issue, не ad-hoc string.

Card rules:

- District Control требует `district_control < workshop`;
- Accountant base requirement: `vp >= 1`; Gray Cardinal bypass — M8 hook;
- Brothel и active Cops нельзя купить повторно;
- active Cartel/Judge нельзя купить повторно;
- depleted Cartel можно reactivate.

Success:

1. Вычти final price.
2. Размести card по Resource type/destination.
3. Для Status добавь documented VP.
4. Добавь ID в `purchased_this_round`.
5. Consume matching modifier.
6. Validate final state.

Placement:

- Engine → engine fields;
- Status → counters + 1/2/3 VP;
- Defense → Cops/Cartel/Judge state;
- War → hand.

Failure оставляет Nal, random, cards, flags и modifiers deep-equal unchanged.

INCOME CONTRACT:

Exact order для каждого player:

1. +2 Nal за Laundry.
2. `SeededRandom.roll_d6_pair`.
3. +1 Nal за Informant.
4. Brothel +5 на doubles.
5. Black Cash меняет bonus на +6.
6. Add total Nal.
7. Process Cops upkeep.
8. Debt hook.
9. Contract hook.
10. Validate result.

Random:

- ровно 2 steps на resolved player;
- updated random state сохраняется;
- preview не rolls dice;
- failure до roll не consumes random.

Black Cash проверяй только через documented contact state. Не реализуй ContactLogic.

Cops:

- active timer увеличивается после income;
- interval 3;
- human interval 2 при Turf Level 5+;
- AI interval остаётся 3;
- due + Nal >= 1: pay 1, timer 0, active;
- due + Nal < 1: timer 0, inactive;
- Nal не становится negative.

Debt/Contract modules ещё не реализованы. Не fake-stubь penalties/progress и не заявляй, что hooks выполнены. При compile-time dependency conflict создай `OQ-*`.

REBUILD CONTRACT:

- success: subtract price, +1 District Control, +3 VP, flag false;
