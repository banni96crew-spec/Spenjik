# Проект: The Turf / Передел

## О чём продукт

The Turf / «Передел» — desktop single-player card/board mini-game
внутри визуальной новеллы.

Игрок и AI управляют районами, зданиями, ресурсами, картами и эффектами.
Игра строится вокруг детерминированных фаз, экономики, рынка и конфликтов.

Текущий этап: Milestone M12 — Turf Levels.
Он выполняется строго после завершённых M0–M11.

## Стек

- Godot 4.6.2 stable.
- GDScript.
- GUT.
- `.tres` Resources.
- Windows/Linux.
- Без web stack.

## Архитектура

- Logic-first: Turf rules находятся в logic layer.
- Data-driven: уровни 0–10 представлены Resources.
- UI является только view/input layer.
- Gameplay не парсит `effect_summary`.
- Turf Levels модифицируют owner systems через чистые helpers.
- Validation выполняется до mutation.
- Helper/preview functions не мутируют state.
- Failed setup/purchase не изменяет state или flags.
- TurfLevelLogic не использует RNG.
- Уровни cumulative: Level N включает эффекты 1..N.
- Turf Level выбирается только при setup и не меняется во время run.

## Договорённости по коду

- Каждый `.gd` файл меньше 250 строк.
- Используй existing state, IDs, Resources и result conventions.
- Setup order: role → strong AI assignment → Turf Levels 1–3.
- Level 2 требует ровно одного заранее отмеченного strong AI.
- M12 не реализует strong AI selection из M13.
- Level 6 действует once per AI per round и consume only on success.
- Level 7 меняет только `strong_ai_victory`.
- Level 8 меняет только три direct upfront payments.
- Level 9 использует strict human VP lead и не мутирует AI Resources.
- M12 предоставляет Level 9 helper, а M13 позже его потребляет.
- Level 10 не использует RNG.
- Resolved fixes `FIX-004`, `FIX-011`, `FIX-013` обязательны.
- Automatic Turf progression из `OQ-002` не входит в MVP.
- Нерешённый вопрос фиксируй как `OQ-*`.
- Ориентиры объёма промтов приблизительные.

## Что НЕ входит в текущий milestone

- Уровни ниже 0 или выше 10.
- Automatic progression между runs.
- Изменение card, role, contract, contact или Street Deal effects.
- Реализация AI system из M13.
- GameStateManager facade из M14.
- Save/load orchestration из M15.
- Turf selection UI из M16.
- Изменение deterministic RNG.
- Runtime mutator для смены Turf Level.
- React, TypeScript, Tailwind, Zustand, Docker, backend и routing.
