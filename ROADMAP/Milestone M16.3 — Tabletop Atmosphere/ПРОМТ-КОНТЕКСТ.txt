## Проект: The Turf / Передел

### О чём продукт

The Turf / «Передел» — desktop single-player card/board mini-game внутри визуальной новеллы.

В матче участвуют:

* 1 human player;
* 3 local AI;
* партия длится 15 раундов;
* финал — Game Over.

Текущий этап: **Milestone M16.3 — Tabletop Atmosphere**.

Это дополнительный UI-polish milestone после M16.1/M16.2 и до M16.4/M17.

Цель M16.3 — сделать игру визуально атмосферной: тёмный криминальный tabletop, грязный стол, единая тема, свет, текстуры, цвета, фон, readable UI без ощущения default Godot/debug interface.

M16.3 **не меняет gameplay rules, card effects, balance, deterministic replay, public facade contracts или state schema**.

---

## Архитектура

Dependency direction:

```text
UI -> GameStateManager -> logic -> catalogs/resources/constants
```

Правила:

* UI является view/input layer.
* Runtime state принадлежит GameStateManager.
* UI вызывает gameplay только через GameStateManager.
* UI не читает и не мутирует `GameStateManager.state`.
* Atmosphere layer не владеет gameplay rules.
* Theme/style/background не должны влиять на state.
* Theme/style/background не должны consume random.
* Gameplay modules не зависят от UI scenes/assets/theme.
* CardView остаётся template-based.
* M15 replay determinism и M16/M16.1/M16.2 UI boundaries должны остаться зелёными.
